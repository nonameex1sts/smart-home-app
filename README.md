# Smart Home App

1. Open Terminal and Clone the Repo

```bash
  git clone https://github.com/Lakhankumawat/smart-home-app.git
```
2. Cd Over 
 
```bash
   cd smart-home-app
```
3. Run Pub
```
   Flutter pub get
```
4. Creating Native Splash
```
  flutter pub run flutter_native_splash:create
```
5. Run App
